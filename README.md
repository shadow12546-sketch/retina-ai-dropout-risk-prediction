# retina-ai-dropout-risk-prediction
ML ensemble model predicting student dropout risk (Low/Medium/High) using attendance, grades, and counsellor notes. Built for the RETINA AI hackathon — stacked ensemble (LightGBM, XGBoost, ExtraTrees, RandomForest, CatBoost) with pseudo-labeling and threshold tuning. Macro F1: 0.7161.
